# JawnRemote (app)

The Flutter mobile app for **JawnRemote** — use your phone as a wireless mouse and
keyboard for your PC over Wi-Fi.

See the [project README](../README.md) for the full picture (PC server, firewall,
Play Store release/signing, protocol).

- Run on a device/emulator: `flutter run`
- Debug APK: `flutter build apk --debug`
- Release bundle: `flutter build appbundle --release`
- Application id: `com.jawnstoninc.jawnremote`
